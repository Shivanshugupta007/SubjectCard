# SubjectCard Backend

This is a Node.js + Express.js backend.

## 1. Clone the Repository

git clone https://github.com/Shivanshugupta007/SubjectCard.git
cd SubjectCard

## 2. Add ENV Variables

PORT            (PORT For Application)
MONGO_URI       (URI For DB Connection)
HF_API_KEY      [Hugging Face API Key To Guess The Subject (Optional)]

## 3. Steps to Set Up & Run This Application

npm install  (Install all The Dependencies)

nodemon app.js  (For Automatic Restart Application On Changes)
